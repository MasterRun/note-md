# linux总是在某些时候发出峰鸣器的声音beep

百度到两种方式
### 临时降低或关闭声音
[参考](https://segmentfault.com/q/1010000004123840)
1. 在命令行下敲入 alsamixer
2. 进入音量调节界面，按 F6
3. 选择主要的声卡，我的是1号声卡
4. 按左右方向键定位到 Beep 那一列
5. 按键盘上的 M 键使之静音; 00 表示生效
我试一下，管不掉，不过声音可以降低不少，不过每次开机需要重新设置
### 修改配置
确保/etc/inputrc中有如下一行未被注释
```set bell-style none  ```
之后注销或者重启试试吧<br/>
